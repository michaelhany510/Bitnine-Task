
This code Implements the 4 arithmetic operations (+, -, *, /) in addition to the fibonacci sequence in the explained question struct.
It acts as a calculator. 
Function Pointers are used to choose which operation to be used at the runtime of the program. 


--------------------------------------------------------------------------------------------------------

Development Environment:

	- Any c compiler can compile this code successfully, I use the latest gcc compiler version

--------------------------------------------------------------------------------------------------------

How to run:
	- Open the code in your IDE, (or compile the file using c compiler)
	- Run The code and outputs will be written at the output terminal.
	- You can edit the code to check outputs for other number values.
