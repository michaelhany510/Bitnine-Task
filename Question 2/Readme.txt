
This code Implements the following function in c:
	
	F(n) = F(n-3) + F(n-2) where F(0) = 0, F(1) = 1, and F(2) = 2.

--------------------------------------------------------------------------------------------------------

Development Environment:

	- Any c compiler can compile this code and run it successfully, I compile using Gcc latest compiler.

--------------------------------------------------------------------------------------------------------

How to run:
	- Open the code in your IDE (or compile the file using C compiler).
	- Run The code and enter the number you need to calculate the sequence at.
